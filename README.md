# Competição do Kaggle de Classificação Binária  
  
  Competição do Kaggle para criar algoritmo de classificação binária com features categóricas. Foi utilizando pacotes em linguagem **R** e *Regressão Logística* para modelar o problema com várias técnicas de tratamento das features.

* [Script usando KNN-5 para missing value usando o modelo de Regressão Logística com  cross-validation.](https://github.com/abelaira/kaggle-categorical-feature-encoding-challenge-II/blob/master/script/Kaggle_Categorical-Feature-Encoding-Challenge-II_RL-plus-MissingValue-KNN-5-CV.R)  
* [Script usando LOCF (last observation carried forward) para missing value usando o modelo de Regressão Logística.](https://github.com/abelaira/kaggle-categorical-feature-encoding-challenge-II/blob/master/script/Kaggle_Categorical-Feature-Encoding-Challenge-II_RL-plus-MissingValue-LOCF-v2.R)  
* [Script usando LOCF (last observation carried forward) para missing value usando o modelo de Regressão Logística com cross-validation.](https://github.com/abelaira/kaggle-categorical-feature-encoding-challenge-II/blob/master/script/Kaggle_Categorical-Feature-Encoding-Challenge-II_RL-plus-MissingValue-LOCF-CV.R)
